### mcp-server-langfuse

https://langfuse.com/docs/prompts/mcp-server

- `git clone https://github.com/langfuse/mcp-server-langfuse.git`
- `npm install`
- `npm run build`

To run the mcp server, can execute `node $ABSOLUTE_PATH/build/index.js`
